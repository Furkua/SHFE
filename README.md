# Superhuman Formulation Engine

App demostrativa en Streamlit para formular barras Superhuman.

## Correr localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Qué incluye

- Hero con logo y título.
- Módulo de ingredientes editables.
- Módulo de fórmula editable.
- Toggle de trabajo:
  - Fórmula → Tabla nutricional.
  - Objetivo nutricional → Fórmula sugerida.
- Motor simple de optimización por logros opcionales.
- Scoring técnico simple.

## Privacidad

Esta versión puede correr localmente. No subas datos sensibles, recetas reales finales, costos reales ni proveedores a repositorios públicos.
